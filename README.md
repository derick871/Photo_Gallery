##  Tech Stack & Dependencies

*   **Frontend Interface & Styling:** HTML5, CSS3, Bootstrap 5 / Tailwind CSS
*   **Media Processing Engine:** Pillow (Python Imaging Library)
*   **Environment Layer:** python-dotenv (Secure credential tracking)
